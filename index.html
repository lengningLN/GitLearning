I am learning Git （2019-05-06）

Terminologys (术语)

Basic Commands (常用命令)

  1. 添加配置

    git config --local/global/system user.name 'Your name'

    git config --local/global/system user.email 'Your email'

    清除，--unset

    git config --unset --local/global/system user.name

    查看配置

    git config --list --local/global/system

  2. 基本操作

    git status : 查看工作区、暂存区的状态

    git add 文件名: 将工作区新建/修改的内容添加到暂存区

    git commit -m "commit message" 文件名: 将暂存区的内容提交到本地库

  3. 查看历史

    git log --all : 查看所有分支历史

    git log --all --graph : 查看图形化历史

    git log --oneline : 查看单行的简洁历史

    git log --oneline -n3 : 查看最近三条的简介历史

    git log --oneline --all -n3 --graph : 查看所有分支最近4条单行的图形化历史(log 之后的命令没有先后顺序，可任意排序)

  4. 分支操作
  5. 比较文件差异

    git diff [文件名] : 将工作区中的文件和暂存区的进行比较

    git diff [本地库历史版本] [文件名] : 将工作区的文件和本地库历史记录比较，不带文件名的话，会比较多个文件

  6. 解决冲突

    1. 编辑文件，删除特殊符号，删除重复内容或者保留内容

    2. 将文件修改完毕后，保存退出。

    3. git add [文件名]

    4. git commit -m"日志信息" (注意此时commit时不能带文件名)
